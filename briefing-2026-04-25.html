🔒 Cybersecurity THREAT INTEL

Critical "Copy Fail" — Linux LPE CVE-2026-31431 Affects All Distros Since 2017

Researchers disclosed CVE-2026-31431 (CVSS 7.8), a local privilege escalation vulnerability dubbed Copy Fail that allows an unprivileged local user to gain root on any Linux distribution shipped since 2017. A 732-byte Python proof-of-concept can edit a setuid binary by writing controlled bytes into the page cache of any readable file. Affected distributions include Amazon Linux, RHEL, SUSE, and Ubuntu. Patches are being coordinated across vendors; status varies by distribution.

Sources: The Hacker News · CISA Weekly Bulletin

Critical Windows Shell Zero-Click CVE-2026-32202 — NTLMv2 Hash Theft via LNK File

Microsoft confirmed active exploitation of CVE-2026-32202, a Windows Shell spoofing vulnerability stemming from an incomplete patch for an earlier APT28 (Fancy Bear) zero-day. When Windows Explorer renders a folder containing a malicious .lnk shortcut, it automatically resolves embedded UNC paths, sending the victim's NTLMv2 authentication hash to an attacker-controlled server — without the user needing to open or click anything. The stolen hash enables NTLM relay attacks and offline cracking for lateral movement. CISA added it to the KEV catalog with a May 12 federal patch deadline.

Sources: The Hacker News · Akamai Research · The Register
