Critical cPanel CVE-2026-41940: CVSS 9.8 Auth Bypass Exploited Since February, 1.5M Servers Exposed

cPanel published an emergency advisory for CVE-2026-41940 (CVSS 9.8) — a critical authentication bypass in cPanel & WHM affecting all versions after 11.40. The flaw exploits CRLF injection in the login process: attackers can inject arbitrary session properties (including user=root) via a malicious basic authorization header, gaining full administrative control over the server and all hosted websites. Rapid7 scans identified approximately 1.5 million cPanel instances publicly exposed. CISA added it to the KEV catalog; Cloudflare issued an emergency WAF rule update. Evidence suggests targeted exploitation began as early as February 23 — two months before the public patch.

Sources: Help Net Security · Rapid7 · watchTowr Labs

High BePrime Mexico Breach: Attackers Control 1,858 Network Devices After MFA Gap

Mexican cybersecurity company BePrime disclosed a breach in which attackers infiltrated administrator accounts lacking multi-factor authentication, gaining control over 1,858 network devices and over 2,600 connected endpoints. The attack leveraged credential stuffing against the company's management console — a sobering incident given that BePrime itself operates as a managed security service provider for enterprise clients. The breach raises significant third-party risk questions for BePrime's client base.

Source: SharkStriker April Breach Roundup

High CISA Warns: CVE-2026-32202 Zero-Click Windows Shell Flaw in Federal Scope

CISA formally added CVE-2026-32202 (Windows Shell NTLMv2 hash theft) to its Known Exploited Vulnerabilities catalog, setting a mandatory May 12 federal remediation deadline. The agency noted particular concern about the flaw's zero-click nature: simply downloading a malicious file to a shared network folder can trigger credential theft across all machines that browse that directory. Enterprises are advised to enforce SMB signing, disable NTLMv1, and block outbound TCP 445 at perimeter firewalls pending patch rollout.

Sources: Help Net Security · CISA KEV
