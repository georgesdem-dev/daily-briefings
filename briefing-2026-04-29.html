Critical cPanel CVE-2026-41940: 44,000 IPs Scanning — Mass Exploitation Underway

The Shadowserver Foundation confirmed that 44,000 unique IP addresses were observed scanning for or actively exploiting CVE-2026-41940, the cPanel CVSS 9.8 authentication bypass, within 24 hours of public disclosure. The flaw allows unauthenticated root-level access via CRLF injection in the session cookie — exposing all websites hosted on vulnerable cPanel instances. CISA has set a May 3 federal patch deadline. Cloudflare's emergency WAF rule blocks known exploit signatures but is not a substitute for patching. Organizations using shared hosting providers should verify patch status with their provider immediately.

Critical Python PyPI Supply Chain: "Lightning" Package Poisoned — Credential Theft Campaign

Threat actors compromised the popular Python machine learning package Lightning (used by tens of thousands of ML engineers) and pushed two malicious versions — 2.6.2 and 2.6.3 — to PyPI. Both versions silently execute credential theft routines on import, targeting cloud credentials, SSH keys, and environment variables. The campaign is assessed as an extension of the "Mini Shai-Hulud" supply chain operation that previously targeted SAP-related npm packages. PyPI has pulled both malicious versions; affected developers should rotate all credentials on any system where Lightning 2.6.2 or 2.6.3 was installed.

Sources: The Hacker News · SharkStriker April Breach Roundup

High Linux "Copy Fail" CVE-2026-31431: CERT-EU Issues High-Priority Advisory

CERT-EU issued a high-priority advisory for CVE-2026-31431 ("Copy Fail"), the Linux kernel local privilege escalation flaw disclosed April 25. A 732-byte Python proof-of-concept is publicly available and grants root on any Linux distribution built since 2017 — including all major cloud-native and container host distributions. Patches are available for RHEL, Ubuntu, SUSE, and Amazon Linux. Containerised environments are particularly at risk: a compromised container can exploit the flaw to break containment and achieve host root.

Sources: CERT-EU Advisory · The Register
