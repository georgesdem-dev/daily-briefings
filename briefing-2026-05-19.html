🔴 Zero-Day CVE-2026-42897 — Microsoft Exchange Zero-Day Actively Exploited, No Patch Yet

On May 14, Microsoft disclosed CVE-2026-42897, a critical zero-day in Exchange Server Outlook Web Access (OWA) that is being actively exploited in the wild. CVSS score: 8.1. The flaw is an improper input neutralization (XSS/spoofing) vulnerability — an attacker sends a specially crafted email, and if the victim opens it in OWA under certain conditions, arbitrary JavaScript executes in their browser context.

Affected versions: Exchange Server 2016, 2019, and Subscription Edition (SE). Exchange Online is not impacted. Microsoft has not yet released a patch, but has pushed an automatic mitigation via the Exchange Emergency Mitigation (EM) Service for organizations with EM enabled. CISA has added it to the Known Exploited Vulnerabilities catalog.
